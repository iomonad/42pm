# 42pm - 42 project manager 
## Fast scaffolding your 42 projects. 
```bash
$ python 42pm -h
usage: 42pm [-h] --projects {fillit} --libft LIBFT --user USER [--ssh]
            --outdir OUTDIR [--list LIST] [--config CONFIG] [--version]

Scaffold 42 projects.

optional arguments:
  -h, --help            show this help message and exit
  --projects {fillit}, -p {fillit}
                        Select project to scaffold
  --libft LIBFT, -l LIBFT
                        Select the libft to use from Github. Ex:
                        <username>/<libftrepo>
  --user USER, -u USER  Your user name.
  --ssh, -s             Use ssh instead https.
  --outdir OUTDIR, -o OUTDIR
                        Relative path to scaffold project
  --list LIST           Show list of available projects.
  --config CONFIG, -c CONFIG
                        Use configuration file.
  --version             show program's version number and exit
```
